# Fresh Bites Bakery 🍰

A simple 5-page bakery website built with HTML + CSS.

### **About**
Fresh Bites Bakery is a responsive website showcasing cupcakes, bread, and bakery products. Made for my web dev assignment.

### **Pages**
- `index.html` - Home page
- `about.html` - About us 
- `products.html` - Our products menu
- `enquiry.html` - Contact/enquiry form

### **Tech Used**
- HTML5
- CSS3
- Responsive design with `welcome-box` container

### **How to view**
Just open `index.html` in your browser. No setup needed.
